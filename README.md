# Comparaci-n-de-Modelos-Supervisados-usando-CV
Este repositorio presenta un análisis comparativo de modelos de regresión (Lineal, Ridge, Lasso y Árboles de Decisión) para la predicción de precios inmobiliarios. Implementa el ciclo completo de ML: preprocesamiento, validación cruzada y evaluación final, con un análisis crítico sobre el sobreajuste y la capacidad de generalización.
